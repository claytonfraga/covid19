Apresentação

Este site tem como objetivo apresentar dados resumidos de casos relatados da COVID-19, e projeções de novos casos para 7 dias utilizando modelos estatísticos.

Procedimento adotado

Os dados são obtidos e processados de duas fontes:

  • Dados disponibilizados pelo European Centre for Disease Prevention and Control (ECDC) são de casos relatados até o dia anterior (RICHIE, 2020).  
  • Outra fonte de dados utilizadas é o Worldometer - Coronavirus, que permite ao público relatar casos da doença, informando: o país, o número de casos e o link para o site de origem para checagem da equipe (WORLDOMETER, 2020).

De posse dos dados provenientes das duas fontes, são realizados procedimento de tratamento com o objetivo de realizar a extração de dados das fontes, a adequação de formatos, a eliminação de eventuais erros de duplicidade, dentre outras. Neste ponto temos um conjunto de dados unificado dos casos relatados da COVID-19 deste o início do surto até a data atual. Importante salientar que nenhum tipo de alteração numérica é realizada, e a responsabilidade pelo número de casos relatados é das fontes consultadas.

Em seguida são realizados ajustes de dois modelos de regressão: um que captura a tendência de crescimento exponencial; e outro, logístico, que leva em consideração o crescimento do número de casos, mas permite a modelagem da saturação da curva.

A escolha dos modelos baseia-se em duas hipóteses:

  1. O exponencial: cuja expectativa é capturar a tendência de crescimento do número de casos considerando uma situação extrema, por exemplo, em um cenário no qual o vírus se espalha com mais velocidade
  2. O logístico: para o qual se espera capturar o comportamento de crescimento menos acelerado, podendo chegar a uma saturação. Este modelo poderia explicar um cenário no qual medidas de higiene, isolamento social, e demais cuidados produzam a desaceleração da transmissão, e como consequência, a redução do número de casos relatados.

Não temos a pretensão de validar as  hipóteses apresentadas.

Por fim, são realizadas projeções do número de casos da COVID-19 para os próximos 7 dias a considerar a data atual, utilizando as equações ajustadas para cada modelo, com o respectivo erro associado ao ajuste da equação, apresentado de forma gráfica.
